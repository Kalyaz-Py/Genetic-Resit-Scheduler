# Система автоматизированного формирования расписания пересдач (WPF / .NET / MS SQL)

Профессиональное десктоп-приложение для оптимизации учебного процесса в колледже. Решает задачу составления бесконфликтного расписания академических пересдач с использованием генетического алгоритма.

## 🚀 Ключевая фишка: Генетический алгоритм
В основе ядра системы лежит кастомная реализация эволюционного (генетического) алгоритма для решения NP-трудной задачи оптимизации ресурсов. Алгоритм учитывает жесткие и мягкие ограничения (Hard/Soft Constraints):
- Конфликты преподавателей (один препод не может быть в двух местах одновременно).
- Конфликты аудиторного фонда (кабинеты).
- Пересечения с основным расписанием занятий групп.
- Баланс нагрузки по дисциплинам и доступным временным слотам.

## 🛠 Стек технологий
- **Язык:** C# (.NET Framework)
- **UI:** WPF (MVVM паттерн, XAML)
- **БД:** MS SQL Server
- **Доступ к данным:** ADO.NET / LINQ
